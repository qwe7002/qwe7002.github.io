寻找一个好用的Markdown编辑器

2017-08-11

由于 SilverBlog 手机客户端并没有提供 Markdown 编辑和草稿功能，所以我们需要找一个不错的文本编辑器来编辑文章和保存草稿。

Markdown X

我在 Play商店 搜索Markdown,最先被发现的是这个应用。这个应用快速便捷，能够通过侧滑屏幕来及时预览文章。同时能够完全输入标题信息和内容。以及网盘同步功能是免费直接可用的。

JotterPad

相对来说功能更多画面更华丽一些的编辑器，号称对诗歌剧本小说写作有帮助。内建有一个词典，但是我觉得并不是非常方便。部分功能需要购买专业版来实现。在预览的排版显示上还不错。可以从SilverBlog的安卓客户端分享写好的文章过去编辑。

锤子便签

锤子出品，必属精品。有质感的拟物化设计的确在这个都是扁平化的时代突显一种独特的气质。这个工具严格来说不是一个 Markdown 编辑器。但是我们可以通过修改编辑模式打开 Markdown 编辑工具栏。有了这个工具栏，在编辑速度上会高出一大档次。毕竟在手机输入法，特别是九键输入法上输入标点符号真的是一件很痛苦的事情。内置的欢喜云也非常不错，可以直接连上同步标签。美中不足的是并没有预览模式和标题选项。不过可以说是非常不错纯粹的内容编辑器了。
